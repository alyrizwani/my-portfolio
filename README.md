# 🌐 Portfolio Website – Ali Raza

A modern, responsive personal portfolio website that showcases creative design skills and frontend development expertise. This project highlights work created using Adobe Creative Suite alongside clean, user-friendly web design principles.

---

## 📌 Overview

This portfolio is designed to present personal information, technical skills, and contact details in a visually appealing and well-structured layout. It emphasizes a balance between creativity and functionality.

---

## ✨ Features

* 📱 Fully responsive design for all screen sizes
* 🎨 Clean and modern UI/UX
* 👋 Hero section with introduction and call-to-action
* 🙋‍♂️ About section with personal background and qualities
* 🧰 Skills section highlighting design and development tools
* 📬 Contact form for easy communication
* 🔗 Social media integration (LinkedIn & GitHub)
* 📱 Mobile-friendly navigation menu

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** – Interactivity and dynamic elements
* **Google Fonts** – Typography (Inter & Playfair Display)

---

## 📂 Project Structure

```
├── index.html      # Main HTML file
├── style.css       # Stylesheet
├── script.js       # JavaScript functionality
├── image.png       # Profile image
├── favicon.jpeg    # Website icon
```

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:

   ```
   git clone https://github.com/your-username/portfolio.git
   ```

2. Navigate to the project folder:

   ```
   cd portfolio
   ```

3. Open `index.html` in your browser.
